Subversion (svn) download and build instructions for vecmath

To build the vecmath package, you must first checkout the vecmath svn repository. For example, run the svn checkout command as follows:

    cd <j3d-root-dir>
    svn checkout https://vecmath.dev.java.net/svn/vecmath/trunk vecmath

After you have downloaded the svn repository, read the README-FIRST.txt file, then read and follow the instructions in the README-build.html file.

Go to the "Subversion" page for instructions on how to access the vecmath source code. Automated svn change messages are sent to the commits 'at' vecmath.dev.java.net list list. Click here to subscribe to this list.
